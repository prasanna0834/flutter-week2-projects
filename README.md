# Flutter Week 2 Projects

This repository contains the Flutter applications completed during **Week 2** of my Flutter Development Internship. These projects focus on Flutter layouts, user interaction, navigation, assets, custom fonts, and package integration.

## Projects Included

### 1. BMI Calculator
A Flutter application that calculates Body Mass Index (BMI) based on user input and displays the result on a separate screen.

### 2. Magic 8 Ball
A simple interactive application that displays random answers when the user taps the button.

### 3. Xylophone
A Flutter application that plays different musical notes when colored keys are pressed.

## Skills Practiced

- Flutter Layout Widgets
- StatefulWidget
- Navigation
- User Input
- Asset Management
- Custom Fonts
- Audio Playback
- Responsive UI
