# Magic Ball

A Flutter application that displays random answers whenever the user presses the button.

## Features

- Random answer generation
- Interactive button
- Image assets
- Dynamic UI updates

## Widgets Used

- MaterialApp
- Scaffold
- Column
- Expanded
- Image
- TextButton
- StatefulWidget

## Concepts Used

- StatefulWidget
- setState()
- Random number generation
- Asset loading

## Getting Started

```bash
flutter pub get
flutter run
```
