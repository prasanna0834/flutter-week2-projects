# BMI Calculator

A Flutter application that calculates the user's Body Mass Index (BMI) based on height and weight inputs.

## Features

- BMI calculation
- Separate result screen
- Responsive layout
- User input handling
- Navigation between screens

## Widgets Used

- MaterialApp
- Scaffold
- Column
- Row
- Expanded
- Slider
- FloatingActionButton
- GestureDetector

## Concepts Used

- StatefulWidget
- Navigation
- Data passing between screens
- Responsive UI

## Getting Started

```bash
flutter pub get
flutter run
```
